# **AccountingWater**
    

 
Данный проект представляет собой iOS-приложение для учета выпитой воды

Обязательные фичи:
* Экран с текущим статусом
* Экран добавления записи
* Работа с локальным хранилищем

Advanced Фичи:
* Нотификации напоминалки
